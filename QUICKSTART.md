# 🚀 Quick Start Guide

<div dir="rtl">

## ابدأ في 5 دقائق!

### الخطوة 1️⃣: المتطلبات

تأكد من تثبيت:
- Docker Desktop ([تحميل](https://www.docker.com/products/docker-desktop))
- Git ([تحميل](https://git-scm.com/downloads))

### الخطوة 2️⃣: احصل على Groq API Key

1. اذهب إلى [groq.com](https://groq.com)
2. سجل حساب مجاني
3. اذهب إلى API Keys
4. أنشئ مفتاح جديد
5. احفظه - ستحتاجه في الخطوة التالية!

### الخطوة 3️⃣: نزّل المشروع

```bash
git clone https://github.com/omarabdhkem/ai-video-studio.git
cd ai-video-studio
```

### الخطوة 4️⃣: التكوين

```bash
# شغل السكريبت التلقائي
./setup.sh

# أو يدوياً:
cp backend/.env.example backend/.env
nano backend/.env  # أضف GROQ_API_KEY الخاص بك
```

### الخطوة 5️⃣: شغّل التطبيق

```bash
docker-compose up -d
```

انتظر دقيقة حتى يبدأ كل شيء...

### الخطوة 6️⃣: استمتع! 🎉

افتح المتصفح وروح على:

**http://localhost:3000**

## 🔍 استكشاف الأخطاء

### المشكلة: "GROQ_API_KEY is required"

```bash
# تحقق من الملف
cat backend/.env | grep GROQ_API_KEY

# إذا كان فارغاً
echo "GROQ_API_KEY=gsk_..." >> backend/.env
docker-compose restart backend
```

### المشكلة: الحاويات لا تبدأ

```bash
# عرض الأخطاء
docker-compose logs

# إعادة البناء
docker-compose down
docker-compose up --build -d
```

### المشكلة: الصفحة لا تفتح

```bash
# تحقق من حالة الخدمات
docker-compose ps

# يجب أن ترى:
# - aivideo-backend    Up
# - aivideo-frontend   Up
# - aivideo-db         Up
# - aivideo-redis      Up
```
